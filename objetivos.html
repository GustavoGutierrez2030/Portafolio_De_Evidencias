Objetivos

Objetivo General

Desarrollar habilidades en la creación de aplicaciones web dinámicas y funcionales utilizando tecnologías modernas, desde el desarrollo en el lado del cliente hasta la integración y uso de herramientas adecuadas para la implementación de sistemas web completos.

Comprender los conceptos del desarrollo web

Comprender los conceptos fundamentales del desarrollo web, incluyendo su arquitectura básica, las tecnologías involucradas y cómo interactúan los distintos componentes para construir aplicaciones web efectivas.

Dominar el desarrollo del lado del cliente

Aprender a desarrollar la interfaz de usuario utilizando tecnologías como HTML, CSS y JavaScript, asegurando que las páginas web sean interactivas, accesibles y responsivas en diferentes dispositivos.

Dominar el desarrolo del lado del servior

Adquirir conocimientos sobre el desarrollo del lado del servidor, utilizando lenguajes y marcos de trabajo como Node.js o PHP, para manejar las solicitudes del cliente, interactuar con bases de datos y gestionar la lógica de negocio de la aplicación.

Intengrar los dos lados y aprender a usar herramientas

Integrar el lado del cliente y del servidor utilizando APIs y otras tecnologías de integración. Además, familiarizarse con herramientas y frameworks modernos como Git, bases de datos y servicios en la nube para asegurar el desarrollo de aplicaciones escalables y seguras.
